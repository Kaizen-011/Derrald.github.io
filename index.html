Mengapa Saya Memilih PPLG

Saya memilih untuk mengikuti Program Keahlian Pengembangan Perangkat Lunak dan Gim (PPLG) karena saya memiliki minat yang besar dalam teknologi, khususnya dalam bidang pengembangan perangkat lunak dan gim. Sejak kecil, saya selalu tertarik dengan bagaimana aplikasi dan permainan diciptakan. Saya ingin memahami bagaimana suatu ide bisa diwujudkan menjadi sebuah produk digital yang berguna dan menyenangkan. Selain itu, dunia teknologi selalu berkembang dengan pesat. Saya melihat peluang yang besar di masa depan dalam bidang ini, terutama dalam pengembangan aplikasi dan gim yang semakin banyak digunakan oleh masyarakat luas. Dengan mengikuti program PPLG, saya yakin dapat memperoleh keterampilan teknis yang dibutuhkan untuk berkontribusi di industri ini. Program PPLG juga menawarkan kesempatan untuk belajar dengan cara yang praktis, tidak hanya teori. Saya ingin terlibat langsung dalam proyek-proyek pengembangan perangkat lunak dan gim, yang menurut saya akan sangat menantang dan memuaskan. Saya percaya bahwa dengan berbekal ilmu yang saya dapatkan di PPLG, saya bisa menjadi seorang profesional yang kompeten di bidang teknologi, dan siap menghadapi tantangan di dunia kerja.

Sudah di Pelajari

Pengalaman Saya dalam Mempelajari Dasar HTML

Saya telah mempelajari dasar-dasar HTML, yang merupakan fondasi dari setiap halaman web. HTML memungkinkan saya untuk memahami bagaimana struktur dan elemen-elemen dasar dari sebuah halaman web dibentuk. Dengan mempelajari HTML, saya sekarang bisa membuat dan menyusun konten pada halaman web secara terorganisir, mulai dari teks, gambar, hingga link. Ini adalah langkah pertama saya dalam memahami bagaimana sebuah website dirancang dan dibangun dari nol.

Pengalaman Saya dalam Mempelajari Dasar CSS

Selain HTML, saya juga telah mempelajari dasar-dasar CSS, yang digunakan untuk mendesain dan memperindah tampilan halaman web. Dengan CSS, saya belajar bagaimana mengatur layout, warna, font, dan berbagai elemen visual lainnya. Saya sekarang bisa membuat tampilan halaman web menjadi lebih menarik dan responsif, sesuai dengan desain yang diinginkan. Pengetahuan ini sangat penting untuk memastikan bahwa website tidak hanya berfungsi dengan baik, tetapi juga menyenangkan untuk dilihat dan digunakan.

Pengalaman Saya dalam Mempelajari Dasar JavaScript

Saya juga telah mempelajari dasar-dasar JavaScript, bahasa pemrograman yang memungkinkan saya untuk menambahkan interaktivitas pada halaman web. Dengan JavaScript, saya bisa membuat elemen-elemen pada halaman bergerak, merespons tindakan pengguna, dan bahkan berkomunikasi dengan server. Ini memberikan saya kemampuan untuk menciptakan pengalaman pengguna yang lebih dinamis dan interaktif, yang merupakan bagian penting dari pengembangan web modern.
